# Threat Intelligence

## Output Format (STRICT)

IOC: <value>
TYPE: IP / DOMAIN / HASH

---

THREAT SCORE: X/100
VERDICT: MALICIOUS / SUSPICIOUS / CLEAN

---

ANALYSIS:
- Reputation summary
- Known activity (botnet, phishing, etc.)

---

MITRE ATT&CK:
- Txxxx: Technique name

---

ASSOCIATED THREATS:
- Malware families / campaigns

---

RECOMMENDED ACTIONS:
- Block / Monitor / Investigate
